## 🎉 Overview

Tired of forgetting items at the store? Say hello to **Shoppy**, your new smart shopping companion! 🚀 This Progressive Web App (PWA) is designed to **revolutionize** the way you shop.

Never forget an item again with **location-based reminders** that pop up when you're near your favorite stores. 🤩 Create and manage multiple shopping lists, add items effortlessly via text, voice, or image, and set store locations for each list. Shoppy sends you a friendly **push notification**, reminding you of exactly what you need when you're in the vicinity.

## ✨ Key Features

- ✅ **Smart Shopping Lists:** Create, edit, and organize shopping lists for all your go-to stores.
- 🎤 **Smart Voice Input:**
    - Real-time speech-to-text conversion
    - Visual recording indicator
    - Live transcription preview
    - Add items hands-free using voice commands
- 📸 **Dual Image Input:**
    - Camera capture for instant photos
    - Gallery access for existing images
    - Live image preview
    - Visual processing feedback
    - Smart error handling
- 📍 **Smart Location Features:**
    - Set store locations using current position or address
    - Customize notification radius (100m-2000m) per store
    - Visual location status indicators
- 🔔 **Enhanced Notifications:**
    - Get precise distance updates
    - Customizable reminder zones
    - Instant alerts when entering store radius
- 🎨 **Sleek Dark Theme:** Enjoy a modern dark theme with T-Mobile pink accents for a visually pleasing experience.
- 📱 **Progressive Web App (PWA):** Install Shoppy on your device for offline access and a native app-like feel.

## 🚀 Tech Stack

- HTML5
- JavaScript ES6+
- Tailwind CSS
- PWA (Progressive Web App)
- Local Storage

## 🛠️ Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/Shawn5cents/Shoppy.git
    ```
2. Open `index.html` in your browser and start shopping smarter!

## 💡 Usage

1. Tap the "✨ New List" button to create a shopping list for your favorite store.
2. Add items using:
    - 📝 Text input
    - 🎤 Voice command
    - 📸 Image upload
3. Watch your lists come to life with emoji indicators:
    - ✅ Mark items as complete
    - 📋 Organize multiple lists
    - 🏪 See nearby store alerts
4. Configure store locations to activate 📍 location-based reminders.
5. Enable push notifications and let Shoppy handle the rest!
